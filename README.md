# Bor és Garnéla Rák ’n’ Roll

Magyar nyelvű, reszponzív egyoldalas éttermi bemutatkozó oldal.

## Indítás

Nyisd meg az `index.html` fájlt egy böngészőben.

## Források és adatok

- Cím, telefonszám és nyitvatartás: nyilvános kecskeméti és éttermi adatlapok alapján ellenőrizve.
- Közösségi linkek: a Bor és Garnéla Facebook- és Instagram-oldalára mutatnak.
- A hero kép eredeti, projekt számára generált vizuális elem (`assets/garnela-hero.png`).
